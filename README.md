# Task 1: CodeAlpha_BasicNetworkSniffer

A Python-based network sniffer developed using Scapy to capture and analyze live network traffic packets. The project displays important packet details such as source IP, destination IP, protocols, ports, and payload data to demonstrate how network communication works.

## Features
- Captures live network packets
- Detects TCP, UDP, and ICMP protocols
- Displays source and destination IP addresses
- Shows port numbers and payload data
- Real-time packet monitoring

## Technologies Used
- Python
- Scapy
